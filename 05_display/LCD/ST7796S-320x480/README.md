# Pin assignments

The table below shows the TFT LCD display pin assignments.

TFT LCD display Pin      | SPI pin | DShanMCU-Mio pin     |  Notes
-------------------------|---------|----------------------|-------------
 PWM                     |         | GPIO7                |
 RESET                   |         | GPIO6                |
 MOSI                    | MOSI    | GPIO11               |
 SCK                     | SCK     | GPIO12               |
 RS                      |         | GPIO5                |
 CS                      | CS      | GPIO10               |
 MISO                    | MISO    | -/GPIO13             |
